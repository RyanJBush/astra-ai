# Astra AI Frontend

React + Vite + Tailwind client for Astra AI.

## Prerequisites
- Node 22+
- Backend API running on `http://localhost:8000` (or configure `VITE_API_BASE`)

## Local development
```bash
cp .env.example .env
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Quality checks
```bash
npm run lint
npm run format:check
```
